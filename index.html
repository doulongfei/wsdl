SOAP Client Tool

[中文版说明]

This repository contains a SOAP client tool with the following key components:

  • cli_caller.py: The main CLI interface for interacting with SOAP services.
  • soap_utility.py: Core SOAP functionality using the zeep library.
  • WSDL files (calculator.wsdl, test.wsdl): Example WSDL files for testing.

Key Commands

Run the CLI tool with the following syntax:

python cli_caller.py <WSDL_SOURCE> <COMMAND> [ARGS]

Example Commands:

  • List methods in a WSDL file: 
    python cli_caller.py test.wsdl list
  • Inspect a specific method to see its signature: 
    python cli_caller.py test.wsdl inspect <METHOD_NAME>
  • Call a method with arguments: 
    python cli_caller.py test.wsdl call <METHOD_NAME> [ARGS]

Debug Mode:

Add the -d flag for detailed logging:

python cli_caller.py test.wsdl list -d

Architecture

  • The CLI (cli_caller.py) handles user input and delegates to soap_utility.py.
  • soap_utility.py uses the zeep library for SOAP operations and direct XML parsing for method signatures.
  • Supports both file-based and URL-based WSDL sources.
  • Implements smart parameter type conversion (string to int/float/file content).
